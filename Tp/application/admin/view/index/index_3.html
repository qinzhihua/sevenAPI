部分数据名称解释

待偿金额

截至统计时点，平台所有借款人尚未偿还的本金总金额（不包括利息部分）

项目逾期率

截至统计时点，当前所有处于逾期状态的项目数与尚未偿还交易总笔数之比

金额逾期率
